SEG KIP AI Platform — ishga tushirish

1) Node.js o‘rnating:
   https://nodejs.org/

2) Papkani oching:
   seg_kip_gpt_connected

3) Terminal/CMD da yozing:
   npm install

4) .env.example faylidan nusxa oling va .env deb nomlang.

5) AI ishlashi uchun .env ichiga API key yozing:
   OPENAI_API_KEY=sk-...
   PORT=3000

6) Serverni ishga tushiring:
   npm start

7) Brauzerda oching:
   http://localhost:3000

Eslatma:
- API key bo‘lmasa ham platforma ochiladi, lekin AI demo rejimda ishlaydi.
- TAG qidiruv lokal bazada API keysiz ham ishlaydi.
- API keyni hech kimga bermang va HTML ichiga joylamang.
